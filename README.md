# E-Commerce Business Intelligence Dashboard

## 📌 Overview

This project is an interactive Business Intelligence dashboard built using Power BI to analyze e-commerce sales data. It provides insights into revenue, customer activity, orders, and product performance through dynamic visualizations and KPIs.

## 🚀 Features

* Total Revenue analysis
* Total Orders tracking
* Total Customers analysis
* Monthly Revenue Trend visualization
* Top Product Categories by Revenue
* Interactive Date and Category slicers
* Data modeling using multiple related tables

## 🛠️ Tools & Technologies

* Power BI
* SQL (MySQL)
* Microsoft Excel
* CSV Datasets

## 📊 Key Performance Indicators (KPIs)

* Revenue
* Orders
* Customers
* Monthly Revenue Trend
* Top Product Categories
* Average Order Value (optional enhancement)

## 🗂️ Dataset

The project uses publicly available e-commerce datasets containing:

* Customers
* Orders
* Order Items
* Products

## 📁 Project Structure

* `PowerBI/` – Power BI dashboard (`.pbix`)
* `SQL/schema.sql` – Database schema
* `SQL/analysis_queries.sql` – Analytical SQL queries
* `Data/` – Source CSV files
* `Screenshots/` – Dashboard images

## 💡 Business Insights

* Identify top-performing product categories.
* Track revenue trends over time.
* Analyze customer and order volumes.
* Enable interactive filtering for deeper analysis.

## ▶️ How to Use

1. Open the `.pbix` file in Power BI Desktop.
2. Load or refresh the dataset if required.
3. Use the slicers to filter by date or product category.
4. Explore KPIs and charts to gain business insights.
